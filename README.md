# Liana — Landing Page

Landing page pour Liana, une API fictive qui unifie l'intégration du Mobile
Money en Afrique centrale (MTN MoMo, Airtel Money, PawaPay) derrière un seul
point d'entrée.

## Contenu

Page unique en HTML et CSS, sans JavaScript.

- Hero avec mockup d'appel API
- Fonctionnalités (3 colonnes)
- Témoignages clients
- Tarifs (3 plans)
- Appel à l'action
- Footer

## Fichiers

- index.html — structure et style (CSS intégré dans une balise style.css)

## CSS

- Variables CSS pour les couleurs, polices et espacements
- Sélecteurs combinateurs (`>`, `+`, `~`) pour cibler des éléments sans
  ajouter de classes supplémentaires
- Pseudo-classes `:nth-child` et `:first-of-type` pour les variations
  visuelles (icônes alternées, plan tarifaire mis en avant, etc.)
- Design responsive avec points de rupture à 860px et 760px

## Utilisation

Ouvrir `Liana` dans un navigateur grâce à son lien "liana-api-82b527.netlify.app", aucune installation nécessaire.
